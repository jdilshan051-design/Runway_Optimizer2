#  AeroRoute: Global Live Fleet Digital Twin Command Center

AeroRoute is an enterprise-grade aviation decision support system. It combines live global flight telemetry with a neural safety matrix to assess landing suitability, calculate aerodynamic stress, and provide automated diversion guidance for complex fleet operations.

##  Key Features

* **Global Fleet Tracking:** Real-time worldwide flight visualization using live satellite telemetry.
* **Neural Risk Engine:** A custom-built MLP architecture that assesses landing safety based on aerodynamic drift and mechanical variables.
* **Autonomous Decision Matrix:** Intelligent rerouting logic that evaluates fuel, weight, and weather conditions to enforce safe landing protocols.
* **High-Contrast Command Deck:** A professional-grade UI designed for mission-critical monitoring and metadata inspection.

##  Technical Stack

* **Frontend:** Streamlit
* **Computational Core:** NumPy
* **Live Mapping:** Leaflet.js
* **Data Source:** OpenSky Network API
